###模板
| 项目   | 内容                                      |
| ---- | --------------------------------------- |
| 用例名称 | *用例名应是一个动词短语，应让读者一目了然地从名字中就可以知道该用例的目标。* |

| 使用语境 | *用例目标，是一个较长的描述，甚至包括触发条件。*               |
| 优先级  | *1-10,开发时适当考虑先后*                        |
| 正常流程 | *正常的用户-系统响应*                            |
| 扩展流程 | *非正常的用户-系统响应*                           |
| 备注   | *备注*                                    |
